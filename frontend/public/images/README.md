# Image Directory Structure

This directory contains all images organized by category:

## Structure

```
images/
├── hero/          # Hero section images
├── about/         # About section images
├── projects/      # Project showcase images
└── skills/        # Skills section images
```

## Usage

Place your images in the appropriate category folders:

- **Hero images**: Landing page hero section backgrounds and graphics
- **About images**: Profile pictures and about section visuals
- **Project images**: Screenshots and thumbnails for each project
- **Skills images**: Icons or graphics for skills section

## Image Formats

Recommended formats:
- **WebP** - Best compression and quality
- **PNG** - For transparent images
- **JPG** - For photographs
- **SVG** - For icons and simple graphics

## Naming Convention

Use descriptive, kebab-case names:
- `hero-background.jpg`
- `profile-picture.jpg`
- `ecommerce-dashboard.png`
- `react-logo.svg`
